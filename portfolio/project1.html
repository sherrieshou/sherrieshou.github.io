Lonely Planet

Interactive and Generative Brand Redesign

{Web Design}

{Brand Design}

{HTML/CSS/Javascript}

Lonely Planet is a periodical introducing travel information. Most of the articles on their official website are about tourist attractions worldwide. To access it, users can simply search by the name of the destination. In my redesign project, I used an API to visualize the user's location and destination by adjusting the position of "Lonely" and "Planet" in the logo.

Coded Skeleton:

The City API database I chose to use records the latitude and longitude data of the cities. In Javascript, I pulled the API data and mapped the two data to the web browser. This tool can not only use for the web experience but also in printed forms.
